Featured Projects

PROJECT 01

The Smart Digital Front Desk

JavaScript / Node.js / Express / REST APIs

Standard AI chatbots frustrate users the second a question gets too complex. I built this widget with a Live Takeover Engine. The AI handles basic FAQs automatically, but if a visitor needs real human help, the system alerts me on my backend dashboard, allowing me to step in and take over the chat live in real-time without refreshing the page.

Real-World Value: Great for customer support desks, medical reception sites, and small business contact pages.
PROJECT 02

Gemini AI Resource Planner

Google Gemini / Python / Interactive Notebooks

Managing project resources—like figuring out tracking hours and staying on budget—involves sorting through massive amounts of data. This interactive workspace allows a manager to type a plain English sentence (e.g., "We are short on hours for Phase 2, how should we reassign the team?"). The Gemini AI reviews team availability, balances the calendar, and suggests an optimized plan in seconds.

Real-World Value: Built for operations managers, development teams, and agency coordinators.
PROJECT 03

Project Resource AI Tool Suite

JavaScript / Text Parsers / Data Cleaning Scripts

A collective set of three lightweight automation utilities built to clean up daily office errors: an Instant Document Searcher to scan large corporate PDFs, an Automatic Report Maker to parse messy incoming team updates, and a Data Mistake Catcher to scan tracking sheets for typos before client presentation.

Real-World Value: Speeds up corporate PMO structures, logistics tracking, and daily administrative workflows.
